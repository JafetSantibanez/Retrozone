# 🎮 RETROZONE

> La nostalgia gamer vuelve a cobrar vida.

---

## 📌 Descripción del proyecto

**RETROZONE** es una plataforma e-commerce enfocada en la venta y exhibición de videojuegos retro, consolas clásicas y artículos relacionados con la cultura gamer vintage.

El objetivo del proyecto es crear una experiencia visual inspirada en los arcades y videojuegos clásicos de los años 80, 90 y principios de los 2000, combinando diseño retro con tecnologías web modernas.

Dentro de la plataforma, los usuarios podrán:

* 🕹️ Explorar videojuegos retro.
* 🛒 Comprar productos y consolas clásicas.
* ⭐ Ver productos destacados.
* 🎮 Descubrir colecciones retro.
* 📰 Consultar noticias y novedades gamer.
* 🔍 Navegar por categorías y catálogo.

# 🚀 Instalación y ejecución

## 1️⃣ Clonar el repositorio

```bash
git clone https://github.com/JafetSantibanez/Retrozone.git

```

## 2️⃣ Entrar al proyecto

```bash
cd RETROZONE
```

## 3️⃣ Instalar dependencias

```bash
npm install
```

## 4️⃣ Ejecutar el proyecto

```bash
npm run dev
```
